# Third-Party Notices

This project vendors a few small third-party assets so it can run offline in a
fullscreen kiosk browser or Android WebView shell without a build step or
network connection.

## Audio Files

The app expects local files at:

- `audio/fajr.mp3`
- `audio/standard.mp3`

Those MP3 files are intentionally ignored by Git. Add your own recordings with
redistribution rights if you plan to publish a fork or packaged release.
